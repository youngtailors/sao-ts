# sao-ts

> Scaffold out a Typescript project.

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao ts my-project
```

### From git

```bash
sao YoungTailors/sao-ts my-project
```
